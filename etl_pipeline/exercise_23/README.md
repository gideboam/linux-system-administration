# Exercise 23 – CSV Data Cleaning ETL Pipeline

## Overview

This exercise demonstrates the implementation of a simple ETL (Extract, Transform, Load) pipeline using Python and Pandas to clean and prepare raw CSV data for further analysis or storage.

Real-world datasets often contain inconsistencies such as duplicate records, missing values, blank rows, and different date formats. This exercise focuses on transforming unstructured and unreliable data into a clean, standardized dataset.

---

## Objectives

The objectives of this exercise are to:

- Extract data from a raw CSV file using Python.
- Identify and remove duplicate records.
- Handle missing values within the dataset.
- Remove unnecessary blank rows.
- Standardize inconsistent date formats into a common format.
- Load the cleaned data into a new CSV file.
